Flower Dodge

What is Flower Dodge?

Flower Dodge is a reskinned take on the Google T-Rex game that is displayed when there is no WiFi. The mechanics are fairly similar to the original, but the sprites and sounds are different. It was created in the spring of 2018 for the course Rich Media Web App Development

Tools

Created in JavaScript using the Canvas API.

Controls

  • space key to jump
  • enter key to play again
