What is Flower Dodge?

Flower Dodge is a reskinned take on the Google T-Rex game that is displayed when there is no WiFi. I got tired of trying to find a version of the game I could play when I had internet, so I created this version. The mechanics are fairly similar, but the main character is different.

Tools

Created in JavaScript

  • enter key to play again
